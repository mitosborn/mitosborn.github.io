Rice University

Data Science Researcher

Houston, TX May 2020 - April 2021

As a researcher in Dr. Sylvia Dee's Climate Lab, I investigated the environmental impacts of the COVID-19 lockdown on Texas. To do this, I utilized statistical modelling on historical data to predict what concentration levels would have been if the lockdown didn't occur in each of Texas' 254 counties. I then divided the data into three partitions (pre-lockdown, lockdown, and post-lockdown) and performed significance testing. I found that the reductions in pollutants from the lockdown were mostly cancelled out by the effects of reopening. In the process of doing this, I created a script using Selenium to automate downloading data. Then, to best communicate these results, I built a web dashboard utilizing the Python package Dash. This dashboard utilizes an interactive heat-map, time series plot, and scatter plot to visualize trends of eight environmental and two socioeconomic indicators throughout the lockdown. To complete this work, I worked closely with a diverse team of environmental science, computer science, and statistics majors.

COVID-19 Environmental Impacts Dashboard

Rice Climate Labortory

Spring 2020 - April 2021 | Dashboard | Github

COVID-19 Environmental Impacts Dashboard

Description: Year long project to assess environmental and socioeconomic impacts of the COVID-19 pandemic on Texas. I worked in a team of 8 students of various majors to design and implement a data science pipeline to assess COVID's impacts. This pipeline collected the most recent environmental and socioeconomic county level data, compared historical and present data using statistics and time-series modelling, and visualized impacts by metropolitan area and county with a dashboard. To implement this pipeline, I first utilized Selenium to automate downloading historical data from the Texas Comission for Environmental Quality (TCEQ) due to a lack of an API. Then, I leveraged Python's Pandas and Statsmodels packages to group data by county and metropolitan area and model what indicator levels would have been if COVID never occurred. Finally, to improve interpretability and simplify analysis of the data, I created an interactive dashboard using Dash, a Python dashboarding package. Packages used: Dash, Plotly, Matplotlib, Statsmodels, Pandas, Selenium.

Spotify Interests Dashboard
Seeking to develop my machine learning and data visualization skills, I developed a dashboard to analyze a user's Spotify music interests and suggest new songs. This dashboard takes a user's Spotify ID as input and compares their song preferences to a dataset of over 10,000 songs. It then uses k-nearest clustering to suggest a list of songs the user should try out. For me, this project was interesting because it attempts to mimic Spotify's music suggestion algorithm. I have always been amazed how accurately Spotify can suggest new songs that I may like. This project made me appreciate how advanced their algorithm is at achieving these results.

Packages used: Dash, Pandas, Sci-Py, Plotly.
